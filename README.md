# NODE.JS API #

Simple Node.JS Project. It include tests, generatid documentation using apiDoc (https://apidocjs.com/), Docker file and Docker Compose.

## API Documentation
Open the API documentation Index file.
```
./apidoc/index.html
```

## Project setup
```
npm install
```

### Start the API
```
npm start
```

### Run Tests
```
npm run test-dev
```
